# readmeGenerator

## Description: 
This generator is a CLI application that takes the user's input to a series of prompts and creates a simple README file.

## Installation: 
Things to have installed: 
- Node
- Inquirer

## Usage: 
- Open the terminal
- Enter 'node index'
- Answer the prompts 
- README file will be found in output folder
